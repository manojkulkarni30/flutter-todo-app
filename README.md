# 🚀 Flutter Todo App

This project is a part of a YouTube tutorial series where we build a Flutter TODO app using Provider for state management and an ASP.NET Core REST API backend.

📺 YouTube Playlist: [📱 Flutter Todo App Tutorial Series | Provider + REST API](https://bit.ly/4pncu52)

📌 Channel: [Manoj Kulkarni](https://bit.ly/41Ryons)

## 🛠️ Tech Stack

- Flutter
- Dart
- Provider
- HTTP
- ASP.NET Core
- Scalar (OpenAPI)

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (Latest Stable)
- Android Studio / VS Code
- Running ASP.NET Core Backend

### Clone Repository

```bash
git clone https://github.com/manojkulkarni30/flutter-todo-app.git
cd flutter_todo_app
```

### Install Packages

```bash
flutter pub get
```

### Run

```bash
flutter run
```

### 🌐 Backend

The Flutter app communicates with an **ASP.NET Core API** backend. The backend source code is available in a separate repository.

🔗 Backend Repository: [https://bit.ly/441WZ8Z](https://bit.ly/441WZ8Z)

### 📖 API Documentation

The backend includes **Scalar**, a modern API reference built on the OpenAPI specification.
After running the backend, open:

```text
http://localhost:5159/scalar
```

to explore and test all available endpoints.
